# Архитектура Uchetgram

> **Полная документация системы** — см. папку [`/docs`](../../docs/README.md) в корне проекта.

## 1. Концепция системы
Uchetgram — это модульная ERP/CRM система, построенная на принципе **Универсальной Сущности**.

### Универсальная Сущность (Universal Entity)
В центре системы находится одна базовая модель данных, которая может принимать различные формы в зависимости от контекста (модификатора).
- **Задача**: Сущность с дедлайном и ответственным.
- **Лид**: Сущность с финансовой ценностью и этапом воронки.
- **Финансовая заявка**: Сущность с суммой, категорией и статусом согласования.
- **Транзакция**: Сущность, представляющая движение денежных средств (банк, 1С).
- **Документ**: Универсальная сущность для хранения файлов, ссылок и внутренних текстов.
- **Задача**: Сущность с дедлайном, ответственным, чек-листом и статусом выполнения.
- **Бизнес-процесс**: Последовательность связанных задач с логикой переходов и ветвлением.
- **Сообщение**: Сущность с контентом и каналом связи.

Это позволяет использовать единые компоненты для отображения, поиска и фильтрации данных по всей системе.

## 2. Структура проекта
Проект организован по модульному принципу:
- `src/components/ui`: Атомарные UI-компоненты (Button, Card, Input).
- `src/components/layout`: Компоненты каркаса (Sidebar, Header).
- `src/modules/*`: Функциональные модули системы (Dashboard, Sales, Finance и т.д.).
- `src/types`: Глобальные определения типов и интерфейсов.

## 3. Технологический стек
- **React 19**: Основа интерфейса.
- **Tailwind CSS 4**: Стилизация через утилитарные классы.
- **Motion**: Плавные анимации и переходы.
- **Lucide React**: Набор иконок.
- **TypeScript**: Строгая типизация всех контрактов.

## 4. Дизайн-система
- **Основной цвет**: `#1356A3` (Primary).
- **Акцентный цвет**: `#F27D26` (Orange).
- **Фон**: `#F8FAFC` (Slate 50).
- **Типографика**: Inter (Sans) для интерфейса, JetBrains Mono для данных.

## 5. Финансовый модуль
Модуль обеспечивает полный цикл управления финансами:
- **Обзор**: Баланс, P&L, распределение расходов.
- **Операции**: Импорт банковских выписок, сверка с 1С.
- **Планирование**: Бюджетирование по статьям (фикс/процент).
- **Заявки**: Процесс согласования выплат сотрудникам.

## 6. Модуль документов
Система управления знаниями и файлами:
- **Файловая структура**: Поддержка вложенных папок.
- **Типы контента**: Внешние ссылки (Figma, Google Sheets), физические файлы (PDF, PNG, XLSX), внутренние текстовые документы.
- **Предпросмотр**: Встроенный просмотрщик для изображений и текстов.

## 7. Модуль задач
Полнофункциональный таск-менеджер:
- **Режимы отображения**: Канбан-доска для визуализации потока и Список для детального анализа.
- **Управление**: Приоритеты (от Низкого до Критического), статусы, теги.
- **Детализация**: Поддержка чек-листов (подзадач), дедлайнов с индикацией просрочки, комментариев.
- **Коллаборация**: Назначение ответственных, отслеживание времени.

## 8. Модуль бизнес-процессов (BPM)
Ядро системы для автоматизации регламентов:
- **Цепочки задач**: Автоматическое создание следующей задачи после завершения текущей.
- **Логика ветвления (Decisions)**: Возможность выбора пути процесса (например, "Одобрить" или "Отклонить").
- **Визуализация Workflow**: Наглядное отображение текущего прогресса, истории шагов и будущих этапов.
- **Ответственность**: Четкое закрепление каждого этапа за конкретным сотрудником.

## 9. Модуль аналитики
Центр принятия решений на основе данных:
- **Финансовая аналитика**: Динамика выручки, расходов и прибыли.
- **Воронка продаж**: Конверсия на каждом этапе, средний чек, LTV.
- **Операционная эффективность**: Скорость выполнения задач, загрузка команды.
- **Визуализация**: Интерактивные графики (Area, Bar, Pie) с использованием Recharts.

## 10. Модуль настроек
Конфигурация и управление системой:
- **Команда**: Управление пользователями, ролями и приглашениями.
- **Интеграции**: Подключение внешних сервисов (1С, Банки, CRM).
- **Биллинг**: Управление подпиской и история платежей.
- **Безопасность**: Настройки доступа и аудит действий.

## 11. Маркетплейс решений
Магазин готовых отраслевых конфигураций:
- **Отраслевые решения**: Готовые наборы настроек для Салонов красоты, Ресторанов, Логистики.
- **Автоматизация**: Пакеты готовых бизнес-процессов (HR, Бухгалтерия).
- **Интеграции**: Коннекторы к популярным сервисам и маркетплейсам.

## 12. Маркетинговый сайт (Landing Page)
Публичная часть системы, описывающая возможности платформы:
- **Hero-секция**: Высокоуровневое позиционирование продукта.
- **Сетка функций**: Детальное описание каждого модуля системы.
- **CTA**: Призывы к действию для конвертации посетителей в пользователей.

## 13. Расширяемость
Для добавления нового функционала:
1. Опишите метаданные в `UniversalEntity`.
2. Создайте новый модуль в `src/modules/`.
3. Зарегистрируйте его в `App.tsx`.
4. Используйте готовые UI-компоненты для сохранения консистентности.
