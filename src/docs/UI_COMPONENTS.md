# Документация UI Компонентов

## Button
Основной компонент действия.
- **Variants**: `primary`, `secondary`, `outline`, `ghost`, `danger`.
- **Sizes**: `sm`, `md`, `lg`, `icon`.
- **Features**: Поддержка состояния загрузки, анимация нажатия.

## Card
Контейнер для группировки контента.
- **Features**: Заголовок, подзаголовок, футер, эффекты при наведении.

## Sidebar
Навигационная панель системы.
- **Features**: Сворачивание, активные состояния, анимация переключения модулей.

## EntityCard
Специализированный компонент для отображения `UniversalEntity`.
- **Logic**: Автоматически меняет иконку и цветовую схему в зависимости от типа сущности.
- **Contexts**: Используется в списках, канбан-досках и лентах активности.

## Header
Верхняя панель управления.
- **Features**: Глобальный поиск, уведомления, профиль пользователя.

## Finance Components
Специализированные компоненты для финансового учета.
- **OverviewTab**: Дашборд с балансом и P&L.
- **TransactionsTab**: Таблица операций с поддержкой сверки.
- **PlanningTab**: Интерфейс для настройки бюджета по статьям.
- **RequestsTab**: Система управления заявками на выплаты.

## Analytics Components
- **MetricCard**: Карточка ключевого показателя с трендом.
- **ChartContainer**: Обертка для графиков Recharts с поддержкой адаптивности.
- **FunnelChart**: Визуализация воронки продаж.

## Settings Components
- **SettingsSidebar**: Навигация по разделам настроек.
- **UserList**: Таблица управления командой.
- **IntegrationCard**: Карточка подключения внешнего сервиса.
- **BillingCard**: Панель управления подпиской.

## Marketplace Components
- **MarketplaceHero**: Промо-блок маркетплейса.
- **SolutionCard**: Карточка готового решения с рейтингом и кнопкой установки.
- **CategoryFilter**: Система фильтрации по типам решений.

## Landing Page Components
- **HeroSection**: Главный экран сайта с CTA.
- **FeatureGrid**: Сетка преимуществ и функций.
- **LandingNav**: Навигационная панель публичного сайта.
- **LandingFooter**: Информационный подвал сайта.
