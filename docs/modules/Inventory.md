# Техническое задание: Модуль склада (Inventory Module)

## 1. Общая информация

### 1.1. Назначение

Модуль склада управляет складами, товарами, остатками и движением товаров.

### 1.2. Сущности

- warehouses — склады
- inventory_items — товары (номенклатура)
- stock_balances — остатки по складам
- stock_movements — приход/расход

## 2. Склады

- name, address
- Список складов, создание, редактирование

## 3. Товары

- name, description, unit (шт, кг, л и т.д.)
- warehouse_id — привязка к складу (или общая номенклатура)
- Остаток вычисляется по движениям

## 4. Движения

- item_id, warehouse_id
- type: in | out
- quantity
- reason (причина)
- date

## 5. Остатки

- Агрегация: сумма приходов минус сумма расходов по item + warehouse
- Таблица остатков с возможностью инвентаризации (будущее)

## 6. API

- GET/POST /api/v1/warehouses
- GET/POST /api/v1/inventory-items
- GET/POST /api/v1/stock-movements
- GET /api/v1/stock-balances

## 7. UI

- Табы: Склады, Товары, Остатки, Движения
- Таблицы с действиями
- Модалки для создания/редактирования
