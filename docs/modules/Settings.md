# Техническое задание: Модуль настроек (Settings Module)

## 1. Общая информация

### 1.1. Назначение

Модуль настроек управляет конфигурацией организации: команда, справочники, интеграции, биллинг, уведомления.

### 1.2. Разделы (табы)

- Профиль и команда — пользователи, роли, приглашения
- Пространство — проекты, статусы, приоритеты
- Отделы — departments
- Финансы — категории расходов, фонды
- CRM — воронки продаж
- Уведомления и автоматизация — настройки уведомлений, правила
- Интеграции — Telegram, банки, 1С, сайты
- Биллинг — подписка, история платежей
- Архив — архивные сущности

## 2. Пользователи

- Список пользователей организации
- Создание, редактирование, архивация
- Роли: admin, employee
- Приглашение по email (будущее)

## 3. Справочники

- Statuses — статусы для задач, сделок
- Priorities — приоритеты
- Departments
- Finance categories
- Sales funnels (воронки и стадии)

## 4. Уведомления

- notification_prefs — для каждого типа события
- telegram_personal, telegram_group
- Типы: new_task, status_change, task_assigned, deal_created, …

## 5. Интеграции

- Карточки интеграций: статус (подключено/нет), кнопка настроить
- Telegram — связка бота с организацией
- Банки, 1С — заглушки под будущее

## 6. Биллинг

- Текущий план (free, business, pro)
- История платежей (из Stripe)
- Кнопка «Сменить план» → редирект на Stripe или модалка

## 7. API

- GET/PATCH /api/v1/settings
- GET/POST /api/v1/users (в рамках tenant)
- CRUD для справочников: statuses, priorities, departments, finance_categories, sales_funnels
