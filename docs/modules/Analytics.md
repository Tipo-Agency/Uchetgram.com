# Техническое задание: Модуль аналитики (Analytics Module)

## 1. Общая информация

### 1.1. Назначение

Модуль аналитики предоставляет дашборды и отчёты: финансы, воронка продаж, эффективность команды.

### 1.2. Разделы

- Финансовая аналитика — выручка, расходы, прибыль, план‑факт
- Воронка продаж — конверсия, средний чек, LTV
- Операционная эффективность — задачи, загрузка, скорость выполнения

## 2. Метрики

### 2.1. Финансы

- Выручка за период
- Расходы по категориям
- P&L (прибыль/убыток)
- План vs факт

### 2.2. CRM

- Конверсия по стадиям воронки
- Средний чек
- Количество сделок
- Дебиторская задолженность

### 2.3. Задачи

- Выполнено за период
- Просроченные
- Загрузка по исполнителям

## 3. Визуализация

- Recharts: AreaChart, BarChart, PieChart
- Карточки метрик с трендом (↑/↓ %)
- Фильтры: период (неделя, месяц, квартал, год)

## 4. API

- GET /api/v1/analytics/finance?period=YYYY-MM
- GET /api/v1/analytics/funnel?period=...
- GET /api/v1/analytics/tasks?period=...

## 5. Компоненты

- AnalyticsModule
- MetricCard — карточка с метрикой и трендом
- ChartContainer — обёртка для Recharts
- FunnelChart — воронка
