# Маркетинговые страницы Uchetgram

## 1. Общая информация

Маркетинговый сайт — публичная часть на том же домене (uchetgram.com). Все страницы доступны без авторизации. Стиль единый: navbar, footer, CTA.

## 2. Основные страницы

| Путь | Название | Описание |
|------|----------|----------|
| / | Главная | Hero, преимущества, модули, отрасли, CTA |
| /about | О компании | Миссия, команда, история |
| /contact | Контакты | Форма, телефон, email, Telegram |

## 5. Общие компоненты

- **Navbar** — логотип, меню (Возможности, Отрасли, Тарифы, …), язык, Войти, Начать
- **Footer** — ссылки (Продукт, Компания, Поддержка), соцсети, ©
- **CTA‑секция** — в конце страниц: «Попробовать бесплатно», «Связаться»

## 6. Роутинг

- React Router. Все маршруты в `App.tsx`.
- **Главная** `/` — компонент `LandingPage` (свой nav/footer и переключение секций).
- **Остальные маркетинговые страницы** — общий `MarketingLayout` (nav + footer + CTA) + страничный контент:
  - Основные: `/about`, `/contact`, `/careers`, `/pricing`, `/investors`, `/solutions`, `/api`, `/help`, `/status`, `/privacy`, `/terms`.
  - Модули: `/modules` (каталог), `/modules/:moduleId` (core, planning, processes, users, finance, reporting, inventory, crm).
  - Отрасли: `/business/:industry` (beauty, medicine, sport, auto, education, retail, services, restaurant).
- Компоненты страниц: `src/pages/marketing/`. Шаблон контента: `MarketingPageTemplate`.
