# Список страниц сайта Uchetgram

## Маркетинговые (лендинг, переключение по currentPage)

Все пять страниц сделаны в едином дизайне: hero, островные секции, чередование свет/тёмный, инфографика, CTA.

| № | Страница        | Описание |
|---|-----------------|----------|
| 1 | **Главная** (home) | Hero, логотипы, USP Telegram, экосистема, цели, как начать, отрасли, статистика, интеграции |
| 2 | **Возможности** (features) | Hero, инфографика «Всё в одном месте», группы (клиенты / финансы / операции), CTA |
| 3 | **Для кого** (industries) | Hero (тёмный), карточки отраслей с кейсами, блок «Один продукт — любые отрасли», CTA |
| 4 | **Цены** (pricing) | Hero, тарифы в $ (Старт $0, Бизнес $29, Pro $79), FAQ, CTA |
| 5 | **Интеграции** (integrations) | Hero, блок «Telegram в центре» (бот, клиенты, CRM), карточки Платежи/Виджет/API/1С, CTA |

## Отдельные маршруты (App.tsx)

- `/about` — О компании  
- `/contact` — Контакты  
- `/careers` — Карьера  
- `/pricing` — Тарифы (дублирует страницу в лендинге)  
- `/investors` — Инвесторам  
- `/solutions` — Решения  
- `/api` — API  
- `/help` — Помощь  
- `/status` — Статус сервиса  
- `/privacy` — Политика конфиденциальности  
- `/terms` — Пользовательское соглашение
